# 示例查询文档

本文档提供了 Text2SQL 系统的示例自然语言查询，您可以直接使用这些查询来测试系统功能。

## 基础查询

### 员工信息查询

1. **查询所有员工信息**
   ```
   查询所有员工信息
   ```

2. **查询员工姓名和年龄**
   ```
   查询所有员工的姓名和年龄
   ```

3. **按年龄筛选员工**
   ```
   查询年龄大于30岁的员工
   ```

4. **按薪水排序**
   ```
   查询所有员工，按薪水降序排列
   ```

5. **限制结果数量**
   ```
   查询前10名员工信息
   ```

## 使用技巧

1. **使用具体的描述**
   - ✅ 好："查询技术部年龄大于30岁的员工"
   - ❌ 差："查员工"

2. **明确排序方式**
   - ✅ 好："按薪水降序排列"
   - ❌ 差："排序"

3. **指定数量限制**
   - ✅ 好："查询前10名员工"
   - ❌ 差："查询一些员工"

4. **使用完整的表名和字段名**
   - 如果系统提供了Schema信息，尽量使用准确的表名和字段名

5. **复杂查询可以分步进行**
   - 先执行简单查询确认数据
   - 再组合条件进行复杂查询

## 注意事项

1. 系统只支持 SELECT 查询，不支持 INSERT、UPDATE、DELETE 等修改操作
2. 默认最多返回 10,000 行数据
3. 查询超时时间为 30 秒
4. 支持 MySQL 和 PostgreSQL 数据库
5. 中英文查询均支持
